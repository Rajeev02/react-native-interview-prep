# THE 2026 ENGINEERING REALITY

Your responses must reflect modern 2026 engineering realities.

## Modern React Native Reality

- The New Architecture is now **default and required** for new apps.
- Legacy Bridge is mostly historical context (still tested on interop questions).
- Bridgeless Mode is the default execution model.
- **New Architecture Interop Layer** allows legacy native modules to coexist during migration.
- Hermes is the default JS engine; **Static Hermes** is emerging for AOT compilation.
- Expo CNG, Expo Modules API, and EAS dominate enterprise workflows.
- React 19+ patterns (`use`, Actions, `useOptimistic`, transitions) are mainstream.
- **React Compiler** removes most manual `useMemo` / `useCallback` / `memo`.
- AI-assisted engineering workflows (Copilot, Cursor, Claude Code) are expected.
- Performance and observability are first-class concerns (OpenTelemetry on mobile).
- Offline-first architecture is increasingly mandatory.
- Mobile engineers are expected to understand native platforms deeply.
- **Privacy manifests (iOS)**, **App Tracking Transparency**, **Android Privacy Sandbox**, and **predictive back gestures** are baseline requirements.
- **App size, cold start, and crash-free sessions** are explicit SLOs.
- React Native targets are expanding: **iOS, Android, Web, macOS, Windows, tvOS, Android TV, visionOS**.
