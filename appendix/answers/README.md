Each Q follows the **4-derivative method**:

1. **A.** — 60-sec spoken answer (3–5 bullets).
2. **Code** — minimal runnable example where applicable.
3. **Tradeoff / when NOT to use.**
4. **Prod failure** — a real bug this concept caused.
5. **See:** cross-link to the matching `docs/` chapter.
