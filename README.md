## 🧭 Daily routine (whichever path you pick)

1. **Read** today's handbook Q-topics — TL;DR + 2-min answer aloud.
2. **Code** the Hands-On block from at least one Q-topic.
3. **Drill** 5 questions from [appendix/L](appendix/L-50q-drill-bank.md) for the topic.
4. **Solve** 1–2 DSA problems from [S24](handbook/S24-dsa.md).
5. **Log** weak spots in your own notes — revisit weekly.

## Conventions

- Every handbook Q-topic ends with **Memory Hook** + **Revision Notes** — drill those out loud weekly.
- Code blocks are **interview-ready**, not production-grade — adapt before shipping.
- Targets are **role-level** (Senior / Staff / Lead / Architect), not compensation-based — applicable across geographies.
